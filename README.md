# OS-KTU

This repository contains materials and resources related to Operating Systems (OS) course from Kerala Technical University (KTU).

## Table of Contents
- **bankers.c**: Implementation of Banker's Algorithm.
- **fcfs.c**: First-Come, First-Served (FCFS) scheduling algorithm.
- **priority.c**: Priority scheduling algorithm.
- **roundrobin.c**: Round Robin scheduling algorithm.
- **sjfs.c**: Shortest Job First (SJF) scheduling algorithm.
- **ipc.c**: Inter-Process Communication (IPC) notes and examples.
- **semaphore.c**: Semaphore implementation and usage.
- **systemcall.c**: Example of system calls in operating systems.
